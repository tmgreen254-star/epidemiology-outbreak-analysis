# epidemiology-outbreak-analysis
This project analyzes real-world infectious disease outbreak data using Python to explore trends in case growth, mortality, and epidemic progression. The goal is to demonstrate how epidemiological time series data can be cleaned, analyzed and visualized to support public health insghts.
## Project Goals 
- Load and clean real outbreak data
- Analyze case and death trends over time
- Calculate key epidemiological metrics
- Visualize infection waves and mortality trends
- Practice healthcare data analysis
## Dataset
This project uses a publicly available outbreak data from:
Johns Hopkins University Center...
The dataset includes:
- Confirmed cases over time
- Deaths over time
- Global country level reporting
## Methods 
- Data cleaning and restructuring
- Calculation of daily new cases
- 7 day rolling averages to help smooth the reporting noise
- Case fatality ratio calculation
- Time series visualization
## Tools Used 
- Python
- Pandas
- NumPy
- Matplotlib
